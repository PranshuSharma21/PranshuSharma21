F.L.O.A.T.

ESP32 microcontroller programmed in Arduino IDE. It reads data from sensors, controls relays for pumps and motors, and connects to Wi-Fi for IoT functions. The robot sends data to a custom web app made using HTML, CSS, and JavaScript, which lets users control the system remotely and see water and battery levels live.

Spyder Byte

Spyder Byte is an innovative, quadrupled spider robot powered by an ESP32, combining cutting-edge robotics with seamless digital control. This project features an LCD display for real-time updates, integrated live camera feed, and a live GPS tracker, all controllable via both an Android app and a web app. The user can effortlessly control the spider's movement, monitor the display, and track its location in real-time, bringing together the worlds of IoT, robotics, and remote interaction into a single immersive experience.

Bio-Drip Revolution Farming

Bio-Drip Revolution Farming is an advanced, IoT-driven solution designed to optimize crop health and water management in smart farming. I developed a live dashboard that tracks key metrics such as crop health, soil moisture levels, and provides a live camera feed for real-time monitoring. The system includes a historical progress dashboard, an automatic tank autofiller, and a water sensor to track tank levels. Integrated with a drip irrigation system, it automatically adjusts irrigation based on soil moisture, ensuring efficient water usage and healthier crops. This project combines precision agriculture with technology to revolutionize sustainable farming practices.
